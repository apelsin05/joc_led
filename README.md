# joc_led

A small Python game involving LEDs. Built on a free afternoon. 

---

## What is this

It's a game. It has LEDs. It runs. 

`assets/` has the visuals. `src/` has the logic. `main.py` starts it. You probably don't need to know more than that.

```bash
pip install pygame  # probably
python main.py
```

---

## Why does this exist

 Mild curiosity about game loops. A low bar for "done".

It served its purpose as a first step into game development before moving on to [HandGestureGame](https://github.com/apelsin05/HandGestureGame), which is what this was quietly warming up for.

Python · probably Pygame · assets that work well enough
